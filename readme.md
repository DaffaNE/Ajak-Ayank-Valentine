# Valentine Card

A simple, playful Valentine mini-site with a cute prompt, animated hearts, and a "Yes/No" interaction.

## Files
- `buatSayangku.html` � Main page and markup.
- `kuyankStyle.css` � Styling and animations.
- `udiMikir.js` � Button behavior and background heart effect.

## How to Run
1. Open `buatSayangku.html` in a browser.
2. Click the buttons to see the interaction.

## Customize
- Update the text in `buatSayangku.html`.
- Tweak colors, spacing, and animations in `kuyankStyle.css`.
- Adjust the logic or effects in `udiMikir.js`.
